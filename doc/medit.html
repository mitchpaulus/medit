medit

A small modal text editor. Vim-like motions, Kakoune-like selection-first editing, multi-cursor.

Modes

ModeEnter from NormalExit
Normal(default)—
Inserti a I A o O cEsc or jf
Ex:Enter / Esc
Search/ or sEnter / Esc

Normal mode — motion

KeyAction
h j k lLeft / down / up / right
H J K LSame, extending the selection
Arrows, Home, EndMove; with Shift extends
w / b / eNext word start / prev word start / word end
W / B / ESame, extending
{ / }Previous / next paragraph
ggTop of file
GLast non-empty line
Ctrl+d / Ctrl+uDown / up 10 lines
xSelect full line(s) under cursor; repeat to extend one line down. Overlapping cursors merge
fx / FxJump forward / backward to char x on this line
tx / TxSame, but stop one short of x
zzCenter current line in viewport

Normal mode — edit

KeyAction
i / aInsert before / after selection
I / AInsert at line start / line end
o / OOpen line below / above
dDelete selection (into register)
cChange selection (delete + enter Insert)
yYank primary selection
p / PPaste after / before
u / UUndo / redo
qQuit

Normal mode — selections & multi-cursor

KeyAction
;Collapse selection (anchor = head)
,Reduce to primary selection
< / >Rotate primary backward / forward
Ctrl+j / Ctrl+kAdd next / previous occurrence as a new cursor
Alt+i objSelect inner text object (see below)
Alt+a objSelect around text object

Text objects (after Alt+i / Alt+a)

KeyObject
wWord
pParagraph
" ' `Quoted string
( )Parentheses
{ }Braces
[ ]Brackets

Search

KeyAction
/Search forward (regex); Enter jumps to first match
sSelect-within: turn matches inside selection into separate cursors
n / NNext / previous match of last search

LSP

KeyAction
gdGo to definition
]d / [dNext / previous diagnostic

LSP starts on demand the first time you open a Go file.

Insert mode

KeyAction
Esc or jfReturn to Normal mode
Arrows, Home, EndMove cursor
Ctrl+a / Ctrl+eLine start / line end (emacs-style)
Ctrl+b / Ctrl+fChar back / forward
Alt+b / Alt+fWord back / forward
Ctrl+wDelete word backward
Ctrl+kKill to line end
Ctrl+uKill to line start
